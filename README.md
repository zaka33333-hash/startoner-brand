# Startoner — Brand Identity

The official brand book for **Startoner** (CHICTRATEC S.L.), a Spanish e-commerce specialising in compatible toner and ink cartridges.

> **Version 1.0** · 27 April 2026

## What's inside

A single self-contained HTML page that codifies:

- **Esencia** — mission, positioning, promise.
- **Identidad legal** — CHICTRATEC S.L. + parent group Recycop (1998).
- **Nombre** — "Startoner" usage rules (one word, no hyphen).
- **Logo** — primary mark + variants + clear-space + don'ts.
- **Sistema CMYK** — six brand colours with hex/rgb/usage.
- **Tipografía** — Manrope display, Assistant body, JetBrains Mono for data.
- **Voz y tono** — "tú" for B2C, "usted" for legal — with do/don't pairs.
- **Componentes signature** — pill button with silver-rotating-glow CTA, tag pills, accent cards, status indicator.
- **Mensajes clave** — the nine quotes that any Startoner surface must defend.

## Open it

```bash
open startoner-brand.html        # macOS
start startoner-brand.html       # Windows
xdg-open startoner-brand.html    # Linux
```

Or host as a GitHub Pages site (free) and share the link with designers, developers, agencies and partners.

## Stack

- Static HTML (no build step).
- Google Fonts: Manrope, Assistant, JetBrains Mono.
- Brand assets served from the live Shopify CDN of star-toner.com.
